1. Abstract

VoteSecure is a full-stack web application designed to modernize electoral voting processes by providing a secure, transparent, and user-friendly online platform. Traditional voting systems face challenges like long queues, voter fraud, logistical costs, and lack of transparency. VoteSecure addresses these through digital voter registration, biometric-free authentication, electronic vote casting (EVM-style), real-time admin controls, and verifiable results.

The system enforces one-person-one-vote using database flags, employs bcrypt password hashing, SQL injection prevention via escaping, and PHP sessions for state management. Built on PHP 8+, MySQL, and XAMPP with a premium glassmorphism UI, it supports complete election lifecycle management (setup → registration → voting → results → reset).

Key innovations include dynamic candidate limits (max 4), election state machine, live vote percentages, and responsive design. Tested end-to-end with 100% pass rate. Suitable for educational institutions and small-scale elections. Future scope includes multi-election support and blockchain verification.

Keywords: Online Voting, EVM, PHP, MySQL, Election Management, Digital Democracy

2. Introduction

2.1 Problem Statement

  • Manual voting: Time-consuming, prone to errors/tampering.
  • No real-time results or transparency.
  • High costs for paper ballots/transport.
  • Voter disenfranchisement due to queues.

2.2 Objectives

  • Develop secure online voter registration and authentication.
  • Implement EVM-style electronic voting with vote enforcement.
  • Provide admin dashboard for election lifecycle control.
  • Display interactive results with charts and winner declaration.
  • Ensure mobile-responsive, accessible UI.

2.3 Scope

Single election per instance, max 4 candidates, unlimited voters. Local deployment (XAMPP). No external APIs.

$voted_already = ($user['voted'] == 1);

if(isset($_POST['vote']) && !$voted_already && $election_status == 'ongoing'){
    // Update votes + mark voted
    mysqli_query($conn, "UPDATE candidates SET votes = votes + 1 WHERE id=$candidate_id");
    mysqli_query($conn, "UPDATE users SET voted = 1 WHERE id=$user_id");
